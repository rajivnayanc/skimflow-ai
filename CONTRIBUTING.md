# Contributing to SkimFlow AI 🤝

We would love to have you contribute to SkimFlow AI! Whether you're fixing bugs, adding new features, or improving documentation, your help is appreciated.

## Getting Started

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/skimflow-ai.git
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Create a new branch** for your feature or bugfix:
   ```bash
   git checkout -b feature/my-new-feature
   ```

## Development Workflow

1. Start the development server using:
   ```bash
   npm run dev
   ```
2. The Vite dev server supports hot module replacement (HMR) for Chrome extensions!
3. Before submitting code, please ensure code quality and run tests locally.

## Submitting Changes

1. Commit your changes with descriptive commit messages.
2. Push your branch to your fork.
3. Open a Pull Request from your fork to the main repository.

## Reporting Issues

If you find a bug or have a feature request, please open a new issue describing the problem and the steps to reproduce it.
